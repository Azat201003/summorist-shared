# Summorist shared

This repository is common for my microservice **summorist** app, that contains next services' repositories: [Users](https://github.com/Azat201003/summorist-users), [Mores](https://github.com/Azat201003/summorist-mores), [Comments](https://github.com/Azat201003/summorist-comments).
Also there's [devops repository](https://github.com/Azat201003/summorist-devops) to run all system or its part.
It contains proto files for grpc and code, generated from it, and may contains common libraries. Also there is mocks for each service.
